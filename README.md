# Ubuntu base image modified for Docker
```shell
docker pull juliolustosa/ubuntu:16.04
```

### What's included?

* Ubuntu 16.04 LTS as base system.
* Unattended Security Upgrades
* Git
* Cron
* Tzdata (Timezone data)
* Curl
* Upstart
* Monit
